
debian_dotfiles - Scripts
----------------------------

A set of Bash scripts for automating the setup and management of Debian-based systems.

│ dotfiles/scripts/         │                                                                 │
│ ├── installs/             │                                                                 │
│ │  ├── Brewfile           │  Packages to be installed via Homebrew                          │
│ │  ├── flatpak.sh         │  Desktop apps to be installed on Linux GUI systems via Flatpak  │
│ │  ├── debian-apt.sh      │  Package installs via apt-get for Ubuntu / Debian-based systems │
│ │  └── prerequisites.sh   │  Cross-distro installation of prerequisite core packages        │
│ └── linux/                │                                                                 │
│    └── dconf-prefs.sh     │  Apply preferences to (mostly GNOME apps) via dconf utility     │
