debian_dotfiles - Scripts for Package Installs
-------------------------------------------------
Each of these scripts is used to automate the installation and updates of packages.
Specific usage instructions can be found in a comment, at the top of each file - read before running.
Files can either be run directly, or can be invoked when you run install.sh.
Be sure to check the contents of any file, and modify to your liking before executing anything.
Currently Debian-based systems are the primary target.
